NAF Security Website

How to use:
1. Upload all files to your GitHub repository.
2. Make sure index.html, style.css, and script.js are in the root folder.
3. Enable GitHub Pages from Settings > Pages.
4. To use your real guard background photo:
   - Put your guard photo in the images folder.
   - Rename it to hero-guards.jpg.
   - In style.css, change:
     url("images/hero-guards.svg")
     to:
     url("images/hero-guards.jpg")

Important things to edit:
- Phone number in index.html
- Email address in index.html
- WhatsApp link in index.html
- Company name if needed
- Gallery photo boxes
